About ProFrame 2D

ProFrame 2D

Version: 1.0.0 (beta)

Developer: Dr. Tahmina Tasnim Nahar & Dr. Md Motiur Rahman, CE, PUST

A professional 2D structural frame analysis tool powered by the Direct Stiffness Method. Features include interactive modeling, node internal hinges, point moments, exact member forces, smooth higher-degree displacement diagrams, dynamic unit conversions, and comprehensive detailed member results.
